# Predictive Maintenance System using Machine Learning

## Overview

Unexpected machine failures in industrial systems can lead to production downtime, increased maintenance costs, and safety risks. 

This project develops a **Machine Learning based Predictive Maintenance System** that analyzes machine sensor data to predict the likelihood of failure before it occurs.

The model learns patterns from historical operating data and identifies machines that are approaching failure conditions.

---

## Objective

The main objectives of this project are:

- Predict whether a machine is approaching failure.
- Estimate machine health condition based on sensor readings.
- Identify important sensor parameters contributing to failures.
- Demonstrate the application of Machine Learning in industrial maintenance systems.

---

# Dataset

## NASA C-MAPSS Turbofan Engine Dataset

Source:
NASA Prognostics Center of Excellence

The dataset simulates the degradation process of aircraft turbofan engines using multiple sensor measurements collected during operation.

### Features include:

- Engine ID
- Operating cycle
- Operational settings
- Temperature measurements
- Pressure measurements
- Fan speed
- Core speed
- Vibration-related parameters
- Other sensor readings

The dataset is used to model machine degradation and predict upcoming failures.
